
    # 1. Какие плюсы у защищенного режима адресации?
	
	Процессы не имеют доступ к областям памяти друг друга
	Увеличение объема виртуальной памяти
	Многозадачность
	4 уровня привилегий, программа не запустит инструкцию в кольце операционной системы
	
	# 2. Что показывает колонка Total в выводе утилиты free?
	
	Общее количество памяти - установленной памяти

	# 3. Для чего нужна shared memory?
	
	Для быстрого обмена информацией между процессорами без необходимости взаимодействования через ядро
	
	# 4. Какие минусы у реального режима адресации?

	Любому процессу доступна вся память компьютера 
	не может использовать 32 и 64-разрядные операции
